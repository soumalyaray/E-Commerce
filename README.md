A modern AI-powered e-commerce platform built to simplify society-based shopping experiences. The application enables users to browse products, manage their cart, complete purchases, and discover products through intelligent AI-assisted search. Designed with a clean and responsive interface, the platform combines modern web technologies with cloud-based infrastructure to deliver a seamless shopping experience.

🚀 Features
AI-powered product search using OpenAI GPT models
Dynamic product catalog with MongoDB integration
Smart keyword extraction for natural language search queries
Shopping cart management system
Checkout and billing workflow
Responsive and modern user interface
Real-time product retrieval through REST APIs
Cloud-hosted database using MongoDB Atlas
Secure environment variable management
Optimized full-stack architecture using Next.js
🧠 AI Search Functionality

The platform leverages OpenAI's GPT model to transform user search queries into optimized product keywords. Instead of relying on exact keyword matches, the AI interprets natural language input and returns relevant products from the database, creating a smarter and more user-friendly search experience.

Example

User Input:

"I need something comfortable for running"

AI Generated Keyword:

Running Shoes

Result:
Relevant products are fetched dynamically from MongoDB.
